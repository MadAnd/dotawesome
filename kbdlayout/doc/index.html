Tables

layouts
List of keyboard layout definitions.

See also:

    layout_definition
layout_definition
Definition of a single keyboard layout. These are put into the layouts table.

Fields:

  • setlayout_args string
  • post_cmd string (optional)

Fields

default_layout
Integer index of a layout from layouts, to be used as a default for new clients.
default_post_cmd
Additional command to be executed after the layout was set. Can be overridden on a per layout basis by defining post_cmd key.

Default: ":", which is essentially a no operation command.

See also:

    layout_definition
cmd_format
Example expansion: { setxkbmap us dvorak && defaultpostcmd }&
icon_dir
Directory containing flag icons. Default: "~/.config/awesome/kbdlayout/flags/"
